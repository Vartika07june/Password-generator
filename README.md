# Password Generator

This project is a Python-based password generator that creates strong and secure passwords using letters, numbers, and special characters.

## Features
- Generates random secure passwords
- User can select password length
- Uses Python random and string libraries

## Technologies Used
- Python
- Random Library
- String Library

## How to Run

1. Clone the repository
2. Open the project in Python or VS Code
3. Run the file

python password_generator.py
